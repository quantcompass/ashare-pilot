# 行情判断方法论

> 本文档沉淀自对个股(以易点天下 301171 为典型案例)的实战分析讨论，
> 提炼成一套**可复用、可优化、可工具化**的行情判断框架。
> **持续迭代** —— 每次新的分析经验都应回流到这里。
>
> 最后更新：2026-06-10

---

## 0. 核心原则(先立规矩)

1. **数据驱动,不拍脑袋**：任何判断/买点都要有数据或回测支撑，拒绝"我觉得"。
2. **诚实标注不确定性**：拿不到的数据就说拿不到，推断就标明是推断，不伪装精确。
3. **不构成投资建议**：本方法论是研究框架，不是荐股；技术面只描述"概率与特征"，不预测必然。
4. **警惕过拟合**：小样本回测"测出来好看、实盘就废"是常态，参数调优要克制。
5. **区分"现象"与"原因"**：技术面能看到"跌了、放量了"，但看不到"为什么"——个股异动多有消息面，务必另行核实。

---

## 1. 分析一只票的标准流程

拿到一只票，按这个顺序过一遍，**逐层递进**：

```
趋势 → 量价 → 独立性 → 阶段 → 资金 → 风险
 ①均线  ②量价配合 ③板块对比 ④波段位置 ⑤主力(若有数据) ⑥消息面/免责
```

每一层的具体方法见下文。

---

## 2. 趋势与信号(均线层)

- **双均线(MA5/MA20)**：快线上穿慢线=金叉(偏多)，下穿=死叉(偏空)；见 `strategy/golden_cross.py`。
- **均线排列**：MA5 在 MA20 之上=多头排列(趋势向上)，之下=空头排列(趋势向下)。
- **局限**：均线**滞后**，金叉死叉在趋势确认后才出现，抓不到头尾、震荡市易反复打脸。
- **判断要点**：先看趋势方向(多头/空头排列)，再看有没有新信号(金叉/死叉)。

---

## 3. 量价分析(最核心的一层)

价格的"质量"由成交量背书。**同样的涨跌，量不同，含义天差地别**：

| 现象 | 含义 | 倾向 |
|---|---|---|
| **放量上涨** | 资金真金白银推动 | 偏多(但要防高位放量见顶) |
| **缩量上涨** | 无量反弹，没有增量资金 | 偏弱(诱多嫌疑) |
| **放量下跌** | 抛压真实、资金出逃 | 偏空 |
| **缩量下跌** | 没人卖也没人买，阴跌 | 中性偏弱 |

**关键判据**：
- **"涨时缩量、跌时放量" = 量价背离向下 = 资金持续流出**(空头特征)。
- **"涨时放量、跌时缩量" = 量价配合向上 = 资金持续流入**(多头特征)。

**实操技巧**：
- **同期对齐对比**：盘中比量能，必须"今天到 X 点" vs "昨天到 X 点"，不能拿半天比全天(否则量比失真)。
- **VWAP 做成本代理**：`VWAP = Σ成交额 / Σ成交量`，某区间 VWAP ≈ 该区间市场平均成本。现价跌破 VWAP 越多，套牢盘越重。
  - ⚠️ VWAP 是**全市场加权均价**，不等于某主力的真实建仓价，只是代理。

**案例(易点天下 6/10)**：放量下跌 -8%、同期量是昨天 1.2 倍，而前一日反弹是缩量 → 判定"昨天反弹是诱多，今天放量杀跌是真出逃"。

---

## 4. 独立行情 vs 板块联动

**一只票跌，要分清是"被板块带的"还是"自己的事"** —— 性质完全不同。

**方法**：用 A股列表的**概念字段**取同概念股，横向对比近期涨跌(见 `peers` 思路)。

- **同概念股普遍跌** → 板块性/系统性，个股是 beta，板块企稳它可能跟着修复。
- **只有它独自暴涨暴跌、板块平稳** → **个股独立行情**(高 beta 独走)，由自身题材资金/情绪驱动，板块不会来"救"它。

**题材资金轮动("击鼓传花")**：
- 同一题材内，资金常按时间在不同个股间轮动拉升：A 涨完 → 撤出 → 去拉 B → 再去拉 C。
- 识别意义：一只题材股放量下跌，可能是**资金跑去拉同题材别的票了**。
- 案例：AI营销/AIGC 题材 5/21(易点天下)→5/27(百纳千成)→6/1(一批)→6/8-9(汉王/博思)的接力。

## 5. 巨量启动波段(四阶段模型)

题材独走股常走出经典的**四阶段波段**，识别当前处于哪个阶段是关键：

```
①地量横盘  →  ②巨量启动  →  ③冲高拉升  →  ④放量派发/回落
 (无人问津)    (量比>3,涨>9%)   (几天大涨)     (顶部放量,跌破成本)
```

**启动信号(已工具化为 `strategy/breakout.py`)**：
- `量比 > 3`(当日量 / 前 5 日均量) **且** `当日涨幅 > 9%`
- 对应 ②阶段，是整个波段的起点。

**阶段标注(用"距启动后高点的回撤"，`breakout.stage()`)**：
- 🟢 距高点 > -3%：拉升/高位(新)
- 🟡 距高点 -3% ~ -12%：见顶回落
- 🔴 距高点 < -12%：已深跌(派发后)

**用途**：扫描题材池(`scan_breakouts`)，找出所有"巨量启动型"波段股并标注阶段——既能找机会(🟢)，也能识别风险(🔴，和已套牢的同款)。

## 5.5 高波动题材波段股专题(A/B 子型)

把易点天下、田中精机这类**"波段巨大"的票归为一类**，并按下跌性质细分——
这是分析这类股的核心方法。

### 归类：什么是「高波动题材波段股」

满足多数特征即归此类：
- **振幅巨大**：年内/120 日「高点 ÷ 低点」≥ 1.8~2 倍(田中精机一年 18→77，超 4 倍)
- **题材驱动**：概念多、与基本面脱节，靠资金情绪炒作
- **独立行情**：与板块联动弱(高 beta 独走，见 §4)
- **量价剧烈**：历史有「巨量启动」(§5)，启动放巨量、派发放量
- **结构清晰**：走"地量 → 巨量启动 → 冲高 → 见顶 → 放量派发 → 下跌"

### 核心方法：先定阶段，再定「下跌性质」(A/B 子型)

**同是这类股，关键看处于波段哪一段、下跌是什么性质** —— 这决定风险与应对。
按下跌性质分两个子型：

| | **A 型：放量派发型** | **B 型：缩量阴跌型** |
|---|---|---|
| 典型 | 易点天下 301171 | 田中精机 300461 |
| 位置 | 刚见顶、派发初期 | 高位腰斩后、下跌中后段 |
| 量价 | **放量下跌**(资金主动出逃) | **缩量下跌**(无人接，温水阴跌) |
| 跌幅 | 还不深(距高点 -10~-20%) | 已深(常腰斩 -40~-60%) |
| 反弹性质 | 诱多反抽，力度大但留不住 | 无量弱反弹 |
| 主要风险 | 下跌刚开始，可能还有大跌 | 跌久跌深，但缩量未必到底，**猜底易套** |
| 应对 | 不追反弹、等转强信号 | 不抄底、等放量企稳 |

### 分析这类股的标准动作

1. **定阶段**：§5 四阶段 + 距高点回撤(腰斩 -50% 是心理位)
2. **定性质**：看量 —— 放量跌(A 型)还是缩量跌(B 型)
3. **独走确认**：§4 同概念对比
4. **风险定级**：这类票风险**最高级**，反弹一律先当**反抽**不当反转；除非放量站回 MA20 上方
5. **不猜底**：A 型等派发完 + 止跌信号；B 型等缩量到极致 + 放量企稳

### 案例对照(2026-06-10)

- **易点天下 301171 = A 型**：5 月巨量启动 → 冲 46.43 → 放量杀跌，距高 -17%，**刚进派发下跌**。
- **田中精机 300461 = B 型**：一年 18→77→44，距高 **-43%(已腰斩)**，**缩量阴跌中后段**。
- → 同为高波动题材波段股，但**一个是"刚开始跌的放量派发"，一个是"跌了很久的缩量阴跌"**，风险点与应对完全不同。**别因为"都跌了很多"就一视同仁。**

## 6. 资金与主力分析(及其硬性局限)

**想判断"主力成本/撤没撤"，必须有资金或筹码数据。没有就别硬编结论。**

| 数据 | 能说明 | 当前可得性 |
|---|---|---|
| 龙虎榜(`dragonTiger`) | 游资/机构席位买卖 | ✅ 可用，但**很多异动股没上榜** |
| 个股资金流向(`moneyFlow`) | 逐日主力/超大单净流入 | ❌ 需升级 stockapi 套餐 |
| 逐笔/Level2 | 大单流向、筹码 | ❌ 付费金刚钻套餐 |
| 筹码分布 | 成本分布 | ❌ 接口不存在 |

**重要判断**：
- **没上龙虎榜 ≠ 没主力**：可能资金分散(非单一坐庄)，也可能主力"分仓控盘"故意规避上榜。**从"没上榜"无法断定是哪种**。
- **VWAP 成本区不是主力成本**：别被"VWAP=42"误导成"跌到 42 主力会护盘"，无证据支持。
- **没有资金数据时，"主力撤多久"无法预测**，只能给带大量假设的量级估算(不可作为交易依据)。

## 7. 买点与回测纪律

**任何"买点"都必须先回测验证，不能凭"看着会反弹"就下手。**

**铁律**：把买点写成明确规则 → 在足够样本上回测 → 看胜率 / 期望 / 盈亏比 / 最大回撤 → 再决定。

**典型教训(反弹买点回测)**：
- 规则："巨量启动→回撤10%→站上5日线买入，持有10日"
- 结果：88 次样本，**胜率仅 51%(≈掷硬币)，平均收益 +1.7%，浮盈+8.8%/浮亏-8.7%**
- 结论：**反弹力度大是真的(浮盈能到 +8.8%)，但没有稳定盈利优势** —— 赚不赚全看止盈止损纪律和运气。
- **启示**：高波动题材的反弹是"诱惑也是陷阱"，大反弹与大回撤是同一枚硬币。

**纪律要点**：
- 参与高波动博弈必须有**硬止盈止损**(如 +5~8% 止盈、-5% 止损)且**严格执行**。
- 警惕**小样本过拟合**：88 次/一年仍可能过拟合，题材退潮后会更差。
- 扣掉手续费+滑点，微正期望往往归零。

---

## 8. 风险与免责(每次分析都要带)

1. **不构成投资建议**，仅技术/量价研究。
2. **技术面看不到原因**：个股独自暴涨/暴跌(尤其 ±7% 以上)多有消息面/资金面驱动，务必核实公告、业绩、行业政策。
3. **盘中数据是"活"的**：当天日线 15:30 后才定稿；盘中拿到的是快照，会变。
4. **数据有局限**：复权基准各源略有差异；高级资金数据受套餐权限限制。
